# What are primitive types? 

Primitive types in Java are the **simplest data type**. Primitive data types are defined by Java; which means 
users **can't** define primitive data types. 

## Eight Primitives: 
- **char** 
- **byte** 
- **short** 
- **int** 
- **long**
- **float** 
- **double** 
- **boolean**
