# How do you instantiate class objects in Kotlin? 

Different from Java, in Kotlin, the keyword "new" 
**does not exsist**. 

## Instantiate classes like so... 

``` kotlin 

class Birds{} 
var birds = Birds() 
val birdies = Birds() 

```
