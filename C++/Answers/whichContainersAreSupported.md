# Which containers are supported? 

## C++ supports sequence containers such as: 
- **array** 
- **vector** 
- **deque** 
- **forward list** 
- **list** 

## C++ supports container adaptors such as: 
- **stack** 
- **queue** 
- **priority_queue** 

## C++ supports associatives containers such as: 
- **set** 
- **multiset** 
- **map** 
- **multimap** 


## C++ supports unordered associative containers: 
- **unordered_set** 
- **unordered_multiset** 
- **unordered_map** 
- **unordered_multimap**
