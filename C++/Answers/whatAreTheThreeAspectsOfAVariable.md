# What are the three aspects of a variable? 

## Aspects 
- **name** 
- **type** (integer, real, character, etc.) 
- **value** 

```cpp 

  #include <iostream>
  
  using namespace std;
  
  
  int main() {
  
    // type name;
    int age;
    bool isDeveloper;
    char a;
  
    // value
    age = 21;
    isDeveloper = true;
    a = 'A';
  
    cout << age << endl;
    cout << isDeveloper << endl;
    cout << a << endl;
  
    return 0;
    /* OUTPUT
    21
    1 (true)
    A
    */
  
  }
  ```
