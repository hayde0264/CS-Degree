
# What is the cout class's purpose? 

The standard class variable **cout** is used to **output data to the console**. The **(<<)** tells the C++ compiler what to output. 


# For instance: 
```cpp 
  #include <iostream>
     
  using namespace std;
     
     
  int main() {
     
    cout << "Hiya \n";
    cout << "This is " << "a " << "message. \n";
     
    return 0;
     
    /* OUTPUTS
    Hiya
    This is a message.
    */
  }          
```
