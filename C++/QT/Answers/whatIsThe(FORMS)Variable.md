# What is the FORMS variable? 

The FORMS variable Specifies the UI files (within dependencies, headers, and source files) that the developer would like to add to the project's build. 

## For instance: 
```bash 
FORMS = somedialog.ui \
    somewidget.ui \
        someconfig.ui
```
