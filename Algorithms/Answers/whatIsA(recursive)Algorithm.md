# What is a recursive algorithm? 


Recursive algorithms solve problems by using functions that call themselves from within their own code. 

## Famous examples: 
- **Factorial of a Number** 
- **Fibonacci Series** 
- **Tower of Hanoi** 
- **DFS for Graph**
